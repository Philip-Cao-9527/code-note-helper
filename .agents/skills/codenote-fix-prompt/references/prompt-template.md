## 填充提示

- `{{版本策略描述}}` 常见写法：
  - `v1.1.3 版本内的最小必要改动修复与治理任务`
  - `v1.1.3 同版本补丁修复任务`
  - `v1.1.4 版本升级前的最小必要收口任务`
- `{{任务类型描述}}` 常见写法：
  - `浏览器扩展页面抓取与回归修复任务`
  - `popup 交互修复与题单治理任务`
  - `题单导入增强与日志治理任务`
- 如果本次只是生成 prompt，不要在模板中假装已经知道根因；根因要求应写成“必须来自 Playwright / 浏览器证据”。
- 如果任务不涉及权限、远程请求、manifest，可以删掉对应专项约束，但保留“不要无故新增权限”。
- 如果任务不涉及版本升级，要明确写“版本号继续沿用 {{当前版本}}，不要误升版”。
