## v1.0.90 - 2026-03-31

## 🎉 CodeNote Helper v1.0.90 发布说明

`v1.0.90` 是一次面向**稳定性 + 授权可用性 + 同步可靠性 + 学习效率**的正式整合发布。  

### 🚀 新增与增强 (New & Enhanced)
- **笔记生成交互升级（LeetCode / TorchCode）**：支持“完整笔记 / 仅答疑”双模式，流式生成过程新增“停止生成”，thinking 与可见答案分区展示，长内容场景下可控性更强。
- **网页端算法私教**：刷题卡壳最崩溃的就是不知道问谁，切出去问AI还得来回贴代码，思路瞬间打断。点击答疑模式直接召唤代码私教。不用离开当前页面，它会顺着你写的源码找bug，讲透刚好卡住的那个点，再给一段能直接跑通的参考代码。哪里不会点哪里，主打陪伴式通关。
- **API 域名动态授权模型升级**：从静态白名单转为按用户配置域名动态申请，新增“前置安全说明 + 浏览器原生授权”双层确认链路，权限范围更可控。
- **笔记页可操作性增强**：顶部关键信息与操作区支持 sticky，长文滚动时“保存/导出/打开题目”等关键操作始终可达。
- **动画与细节体验优化**：猫猫动效改为更平滑的渲染方式，减轻卡顿感；笔记与渲染细节进一步打磨。

### ✨ 修复与体验优化 (Fixes & Improvements)
- **Cloud Sync 功能已移除**：`v1.0.90` 正式版不再提供 Cloud Sync 同步能力，避免多通道状态冲突与历史回滚问题。
- **同步策略收敛**：同步链路明确为“本地存储 + 你主动配置的备份方案”，减少隐式自动同步带来的数据不确定性。
- **首次安装授权稳定性修复**：修复首次授权链路偶发“未授权/无法检查权限”问题，完善 runtime 异常识别与可操作提示。
- **多入口一致性**：弹窗与 options 页统一复用同一套 API 域名授权逻辑，减少行为分叉与排查成本。


### ⚠️ 重要提示：权限与同步使用建议
- **API 域名授权**：首次保存新域名时会出现前置说明与浏览器权限弹窗；拒绝授权将不会保存该配置。
- **Cloud Sync 已下线**：若你曾在历史本地版本中使用过 Cloud Sync，请以当前版本实际设置项为准，不要再按旧流程操作。

### 📦 升级建议
- 升级后建议先在“设置与同步”检查当前可用的同步与备份配置是否符合你的预期。
- 若使用页面内 API 生成，建议确认 `API Base URL / API Key / Model` 已保存且对应域名已授权。
- 若你通过 GitHub Releases 离线安装，建议下载并替换为本版本完整包，避免历史脚本残留带来的行为差异。

### 🛠️ v1.0.90 补丁（2026-04-02）
- **修复 LeetCode 提交通过自动入库误触发**：当题目记录已存在该题时，自动入库动作会被前置拦截，不再触发 `submission_passed` 写入，也不再显示“检测到提交通过”成功提示。
- **修复短时状态切换误报**：提交状态快速切换（通过后紧接错误）场景下，不再因旧状态残留导致误报成功。
- **移除 Markdown `==text==` 黄色高亮语法**：渲染器不再把 `==text==` 转换为 `<mark>`，普通文本保持原样，代码块内内容保持原样。
- **影响范围**：LeetCode 笔记弹窗 API 渲染、TorchCode 笔记弹窗 API 渲染、Notes 页面预览（共用 `window.MarkdownRenderer`）。
- **验证结果**：`node --test tests/**/*.test.js` 通过（78/78）；Playwright MCP 证据见 `docs/playwright-v1.0.90/`。

## v1.0.80 - 2026-03-23

## 🎉 CodeNote Helper v1.0.80 发布说明

`v1.0.80` 是一次面向**稳定性 + 可用性 + 学习效率**的版本升级。  
继续围绕「算法刷题 / 深度学习复盘 / AI 长对话管理」三条主线，强化了从“做题”到“沉淀”的完整闭环体验。

### 🚀 新增与增强 (New & Enhanced)
- **笔记生成模式升级**：支持更灵活的“完整笔记 / 快速答疑”使用场景，满足系统化复盘与即时解惑两类需求。
- **题目入库链路增强**：题目记录入口更顺滑，减少“必须走固定按钮路径”带来的使用阻塞。
- **题单管理体验提升**：题单导入、筛选与进度展示更清晰，便于持续追踪学习路径。
- **时间轴预览能力增强**：长对话场景下的节点预览信息更完整，定位历史关键内容更高效。

### ✨ 修复与体验优化 (Fixes & Improvements)
- **设置页交互改进**：API 与同步配置的“未保存提醒”更清晰，减少误操作导致的配置丢失。
- **笔记渲染细节修复**：标题层级、符号显示、流式内容展示等细节进一步优化。

### ⚠️ 重要提示：坚果云同步避坑指南
本扩展支持将数据完整备份到坚果云 WebDAV。  
**请注意：** 若你在云端已有历史备份，在新设备首次配置后，**请优先点击「从云端恢复」**。  
不要先点「立即备份」，否则可能用本地空数据覆盖云端历史数据。

### 📦 升级建议
- 建议从旧版本升级后，先进入「设置与同步」检查一次同步配置状态。

## v1.0.70 - 2026-03-15

## 🎉 CodeNote Helper v1.0.70 发布说明

这是 CodeNote Helper 的首个正式开源发布版本！🎉 
一个专为算法刷题、深度学习和 AI 学习场景打造的浏览器扩展。拒绝机械的复制粘贴，帮你自动化生成高质量复盘笔记、轻松驾驭超长对话，让刷过的每一道题都成为真正的技术资产！

### 🚀 新特性 (New Features)
* **算法刷题笔记助手**：在 LeetCode / CodeFun2000 题目页面一键生成结构化 Markdown 笔记，包含思路分析、逐行代码讲解与复杂度分析。
* **深度学习笔记助手**：支持 Deep-ML 与 TorchCode 页面，自动识别网络结构题型并生成面试深挖考点。
* **AI 对话时间轴**：在 ChatGPT、Claude、Gemini 侧边栏自动生成可视化时间轴，支持点击跳转、星标收藏与悬停预览，轻松驾驭百轮超长对话。
* **多模型直连**：支持配置所有兼容 OpenAI 格式的 API（如 DeepSeek、Moonshot 等），在题目页面内流式生成解答，所见即所得。

### ✨ 优化与体验提升 (Improvements)
* **灵神题解置顶推荐**：算法笔记生成时，若检测到灵茶山艾府的题解，将自动置顶推荐，引导学习高质量实现。
* **题单进度追踪 UI**：支持一键导入 Hot100、面试经典 150 等题单，完成题单时会触发庆祝动画。
* **深色模式完美适配**：所有注入的悬浮按钮与时间轴组件均支持跟随系统或网页主题自动切换深/浅色模式。

### ⚠️ 重要提示：坚果云同步避坑指南 (Important Notes)
本扩展支持将数据完整备份至坚果云 WebDAV。
**注意：** 如果你此前已经在云端拥有备份数据，在新设备上登录坚果云后，**请务必点击「从云端恢复」**！切勿直接点击「立即备份」，否则本地的空数据会覆盖掉你云端的历史记录。
